RAM.py Examples

Here are some real examples showing how to use RAM.py.

🔥 Basic Example

import ramtools

ramtools.start("YOUR_PROJECT_PATH")

📁 File Organizer

from ramtools import filetools

filetools.organize("./Downloads")

⚡ Fast Processing

from ramtools import speed

numbers = [1,2,3,4]
result = speed.superboost(numbers)
print(result)
