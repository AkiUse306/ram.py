📌 Introduction

RAM.py is a modern Python toolkit that helps you write faster, cleaner and more efficient code.

📥 Installation

pip install ramtools

🚀 Quick Start

import ramtools

ramtools.start("YOUR_PROJECT_PATH")

📂 Features

  • Fast processing utilities - Optimized data processing with superboost() for lightning-fast performance. Process lists, dicts, and custom data structures with minimal overhead.
  • Cleaner file management - Organize files automatically by type (images, documents, videos, code). Clean up downloads and manage directories effortlessly with organize() and clean_downloads().
  • Built‑in debugging helpers - Advanced debugging with trace(), log_info(), and stack trace utilities. Profile your code and track function calls in real-time.
  • RAM‑efficient data tools - Process, filter, transform, and flatten complex data structures. Merge multiple datasets and convert between formats (JSON, CSV, etc.).
  • Seamless Library Integration - Pure Python implementation with zero external dependencies. Works with any Python project and integrates smoothly with existing code.
